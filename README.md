# Sparkly goggles

From scratch:

```bash
make build
make run
```

Then

```bash
make
exit
```

Then

```bash
make reinstall  # presuming firmware exists
```

## Pins

* Yellow wire from reset button - pin 3
* Mauve wire from key - pin 0
* Green data wire - pin 1

Tilter pins

* Green wire - pin 8
* Mauve wire - pin 9
